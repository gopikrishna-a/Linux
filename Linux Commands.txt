$ sudo passwd root

Create new Directory
$ mkdir directoryname

Remove Directory
$ rmdir directoryname

$ mkdir -p /tmp/amrood/test

$ rmdir dirname

Create a new file
$ touch filename

Rename File
$ mv oldfilename newfilename

$ pwd

$ ls

$ ls -la
